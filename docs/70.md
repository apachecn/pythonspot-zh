一个 wxPython 应用程序

在本教程系列中，您将学习使用 wxPython 模块进行[图形（GUI）编程](https://pythonspot.com/gui)的基础。 它是跨平台的：应用程序可以在 Microsoft Windows，Apple OS X 和大多数 Unix 系统上运行。

使用 **wxPython** 制作的应用程序具有本机系统界面，而无需更改代码。
