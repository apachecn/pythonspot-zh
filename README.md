## 下载

### PYPI

```
pip install pythonspot-zh
pythonspot-zh <port>
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g pythonspot-zh
pythonspot-zh <port>
# 访问 http://localhost:{port} 查看文档
```
